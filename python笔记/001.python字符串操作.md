## 字符串方法
```python

s.strip()：               移除字符串头尾指定的字符（默认为空格或换行符）或字符序列
s.upper()：               将字符串小写变大写的  
s.lower()：               将字符串大写变小写的  
s.split(str="")：         以str为分隔符切分  ，返回列表
s.replace(str1,str2)：    str1替换为str2  
s.startswith(str)：       检测开头是否正确  
s.endswith(str)：         检测结尾是否正确  
s.encode()：              改变编码  
s.count()：               计数
s.index()：               找下标
s.isdigit()：             包含数字返回True，否则False
s.join(seq)：             以s作为分隔符，将seq中所有元素合并成一个新的字符串
s.maketrans(str1,str2)：  str1映射转换成str2
s.find(str)：             检测str是否在字符串中，beg和end指定范围，
                          存在返回开始的下s.标，不存在返回-1

```
