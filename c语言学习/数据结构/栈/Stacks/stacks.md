# 栈(stacks)
### 介绍
1. 栈也被称为(last in first out)LIFO,也就是**后进先出**。
2. **只能从栈顶开始访问**，任何元素必须从栈顶开始插入或删除
3. **栈是一个列表或者集合**，它有这样一种限制，**插入和删除元素**只能**从**一端进行，我们称为**栈顶**。
### 基本操作
**时间复杂度都是O(1)**
1. push(栈顶插入元素)
2. pop(栈顶移出元素)
3. top(返回栈顶元素)
4. IsEmpty(检查是否为空)
### 应用场景
1. 帮助我们执行函数(eg.递归调用)
2. 编辑器的撤回操作
3. 编译器(验证括号是否匹配)
